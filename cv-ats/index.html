Regles d'or pour un CV ATS parfait

  • Utilisez des titres de section standards : Experience, Formation, Competences
  • Reprenez les mots-cles exacts de l'offre d'emploi
  • Format PDF texte uniquement (jamais de scan ou image)
  • Police classique : Arial, Calibri, Helvetica, taille 10-12pt
  • Structure lineaire sans colonnes ni tableaux dans le corps
  • Nommez le fichier : Prenom_Nom_CV.pdf
  • Evitez les en-tetes et pieds de page complexes
  • Pas d'images, photos ou graphiques dans le corps du CV
